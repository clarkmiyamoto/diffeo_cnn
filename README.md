# diffeo_cnn

# Installation

HPC Environment
```
module purge
module load python/intel/3.8.6 
```

Activate python environment
```
source diffeo_cnn_clark/bin/activate
```

Pip install
```
pip install -r requirements.txt
```

# SBATCH Example

```
module purge;
module load python/intel/3.8.6;
source diffeo_cnn_clark/bin/activate;
```
